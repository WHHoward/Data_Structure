线性表组目前只支持一个且上限为100（如果可能的话还会改进吧）
重点介绍 16 ～ 24：
16： 加载当前线性表到你指定的文件（目前只支持文件里只有一个线性表）
17： 从你指定的文件里读取第一个线性表到当前线性表（只支持文件里只有一个线性表） 
18： 建立一个空表到线性表组
19： 将当前线性表写到线性表组里
20： 定位线性表
21： 移除线性表
22： 遍历线性表组
23： 将当前线性表切换至指定的线性表
24： 将当前线性表的内容更新至指定的线性表（其实我也想做到不用更新就能修改线性表但精力有限）
